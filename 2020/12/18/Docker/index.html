docker 安装mysql

  1. docker pull mysql:5.7 冒号后加版本，在https://hub.docker.com/_/mysql查看

  2. 启动

    • -p设置端口及映射端口 linux端口:docker容器端口
    • –name 设置容器名称
    • -v设置挂载
    • -e MYSQL_ROOT_PASSWORD=root 设置MySQL密码
